 Unscramble Word Game 🎮

A simple word unscramble game built using HTML, CSS and JavaScript.

 Features
- Random word generation
- Auto shuffle algorithm
- Score system
- Enter key support
- Auto next word


 Technologies Used
- HTML
- CSS
- JavaScript

 How to Play
1. Click Start Game
2. Guess the correct word
3. Press Enter or Check
4. Score increases for correct answers
